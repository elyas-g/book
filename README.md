# book

book is simple application that use html, css and js to build simple library.It simply used to help you manage your library.i

Features:

-Add Books: Users can add new books by entering the title, author, and number of pages.

-Remove Books: Delete any book from your library with a single click.

-Toggle Read Status: Mark books as read or unread.

-Dynamic UI: Updates the library display automatically when actions are performed.

Technologies Used:

-HTML5 – for structure

-CSS3 – for styling and layout

-JavaScript – for DOM manipulation and interactive functionality
